## Run locally

Clone the project and access the folder.

```bash
$ git clone
$ cd (folder)
```

Follow the steps below:

```bash
# Install the dependencies
$ yarn or npm i
# Start the project
$ yarn dev or npm run dev
```

The app will be available for access on your browser at http://localhost:3000
